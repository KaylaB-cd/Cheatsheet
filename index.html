HTML tags

Table Tags

Tag name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Display - flex or inline-flex

justify content to position items left to right
flex-start items positioned in order from left, no extra space before or between
flex-end items positioned in order from right, no extra space before or between
center all items in center with no extra space
space-around items positioned with equal space before and after each item
space-between equal space between but no extra space before or after

Display - flex or inline-flex

Align items (up and down)
flex-start elements positioned at top of parent container
flex-end elements positioned at bottom of parent container
center elelements centered halfway between top and bottom
baseline the bottom of the content of all items will be aligned with each other
stretch if possible, the items will stretch from top to bottom of the container (depending on min/max-height)

Align-content (multiple rows)
flex-start all rows positioned at top of parent container
flex-end all rows positioned at bottom of parent container
center all rows positioned at the center of the parent element with no space inbetween
space-between all rows will be spaced evenly from top to bottom with no space above or below the top and bottom row
space-around all rows will be spaced evenly from top to bottom with the same amound of space
stretch if minimum height or no height is specified, the rows of the elements will stretch to fit the parent container

CSS Grids

Grid properties. display: grid
grid-template-columns eg : 100px 200px; to create 2 columns of different widths
grid-template-rows eg: 50pc 100px 200px; to create 3 columns of different heights
grid-template to state rows then columns in one line. rows / columns
grid-template-area to state where rows / columns start and end
row-gap / column-gap / gap the space between rows and columns (gutters)
grid-row-start / grid-row-end can use span to span a number of rows
grid-column-start-grid-column-end can use span to span a number of columns
grid-area shorthand is grid-row: row start / column start / row end / column end
