HTML tags

Table Tags

Tag name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Display - flex or inline-flex

justify content to position items left to right
flex-start items positioned in order from left, no extra space before or between
flex-end items positioned in order from right, no extra space before or between
center all items in center with no extra space
space-around items positioned with equal space before and after each item
space-between equal space between but no extra space before or after

Display - flex or inline-flex

Align items (up and down)
flex-start elements positioned at top of parent container
flex-end elements positioned at bottom of parent container
center elelements centered halfway between top and bottom
baseline the bottom of the content of all items will be aligned with each other
sstretch if possible, the items will stretch from top to bottom of the container (depending on min/max-height)
